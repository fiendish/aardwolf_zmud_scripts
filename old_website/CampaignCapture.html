Fiendish's Super Amazing Campaign Capture Script For zMUD

Tested to work in zMUD 7.21 on Windows XP. Not guaranteed to work in other versions or system configurations.
What this script does:
  • Captures your campaign mob list to a window that updates as you kill them.
  • Highlights your campaign mobs when you use scan, so that you can track them down more easily.
New in the September 01, 2008 update:
  • The campaign window should now pop up automatically if you close/re-open zMUD in the middle of a campaign.
Download the script here: Campaign Capture Script
IF YOU DO NOT INSTALL THIS SCRIPT AS DESCRIBED BELOW YOU WILL BE SORRY.

------------ BEGIN INSTRUCTIONS ------------

This script installs a class named "CampaignStuff" with a class called "CampaignCapture" inside it.
If any of your scripts conflict with this class structure, they must be removed first or they'll be removed for you.

Do not install this script while in the middle of a campaign. Either finish or clear your campaign before installing.

To install the script go to the Actions menu in zMUD and choose "Execute script..."


Then select the text file containing the script, in this case CampaignCapture.txt, and click Open
