Fiendish's Automated Script Installer...uhh...Script

Tested to work in zMUD 7.21 on Windows XP. Not guaranteed to work in other versions or system configurations.

What this script does:
  • Provides an alias that when given the right parameters automatically downloads and installs a script file.
What this script does not do:
  • Fail silently. If someone can figure out how to make VBScript fail without throwing up a message, let me know, eh?

Download the script here: ScriptInstaller.zip


------------ BEGIN INSTRUCTIONS ------------
If you have an old version already installed, then you can probably just use that one to install the new version. Isn't that neat?

This script installs a class named "ScriptInstaller" containing an alias called "scriptinstall".
If any of your scripts conflict with these names, they must be removed first or things could break.

There is no special procedure necessary for this install. Either import the text file, execute it, or copy/paste the contents.

------------ HOW TO USE THIS SCRIPT ---------

scriptinstall "<complete url>" "<name of script>"
example...
scriptinstall "http://www.fakeurl.com/directory/MySuperAmazingScript.txt" "My Super Amazing Script"

If your URL contains a ~ (like http://www.fakeurl.com/~userdirectory/), you must use two of them (like http://www.fakeurl.com/~~userdirectory/)
because otherwise zMUD will strip it out.


------------- THIS IS IMPORTANT -------------
Make sure you use the real file URL. No redirects. No virtual domains. No framesets.
And test everything to make sure it works.
