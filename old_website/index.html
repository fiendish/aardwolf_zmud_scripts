Fiendish's old zMUD scripts for Aardwolf

Tested to work in zMUD 7.21 on Windows XP years ago. Not guaranteed to work anymore or in other versions or system configurations.

Useful Scripts written by me:
  • Floating AutoMap Capture (Puts the automap into its own floating window.) - Last updated: July 1, 2008
  • Self Updating Dynamic Speedwalks (You'll never have to remember or update a speedwalk alias again.) - Last updated: November 12, 2010
  • The Bigmap Viewer (Shows the bigmap updated as you move around. See page for demonstration video.) - Last updated: September 12, 2008
  • HP/Mana/Moves/Enemy Status Gauges (Graphical gauges for your current status) - Last updated: August 29, 2008
  • Channel Capture (Capture channels to a separate window.) - Last updated: September 14, 2008
  • Campaign Capture (Capture campaign mobs to a separate window.)
  • Speech Transmogrifiers (These scripts alter what you say in interesting ways)
Less useful, but still interesting, scripts written by me:
Note, these are here only for other script-writers to get ideas. The voting aliases no longer work.
  • Browserless Web Voting Aliases (Vote for Aardwolf on TMS and TMC without launching a browser)
  • Automatic Script Installer (Makes it possible to run a script directly from the internet without downloading it first)

Scripts not written by me:
  • yashiro's_spellup_script-V3.zip
